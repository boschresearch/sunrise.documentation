# The SUNRISE Framework

![SUNRISE Logo](img/sunrise_logo.svg){: style="height:240px"}

The **SUNRISE** framework (**S**calable **Un**ified **R**ESTful **I**nfrastructure for **S**ystem **E**valuation) defines a uniform approach to access simulation technologies and boost efficiency of system evaluation by leveraging decentralized compute resources for deployment of execution workloads. The solution is based on a strong modularization and connection of the components via APIs.

*For formal information see [**About**](about.md).*


## Getting Started
A **demonstrator** to experience the basic functionality of the SUNRISE framework is available [here](https://github.com/boschresearch/sunrise.demonstrator).
